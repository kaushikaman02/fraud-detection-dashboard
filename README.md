Fraud Detection Dashboard

📌 Project Overview

This project is a Machine Learning based Fraud Detection Dashboard built using Streamlit.

The dashboard helps identify fraudulent transactions using transaction details and SHAP explainability.

---

🚀 Features

- Transaction Explorer
- Fraud Prediction
- Live Risk Score
- SHAP Waterfall Plot
- Plain-English Explanation
- Searchable Transaction Table
- Interactive Dashboard

---

🛠 Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- SHAP
- Matplotlib
- Plotly

---

📂 Project Structure

app.py
model.pkl
requirements.txt
train_transaction.csv
README.md

---

▶️ Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py

---

🌐 Deployment

This project is deployed using Streamlit Community Cloud.

📊 Dataset

IEEE-CIS Fraud Detection Dataset
